# Telegram TODO Bot (aiogram)

Небольшой Telegram-бот на `aiogram` для ведения списка задач.

## Возможности

- Добавление задач обычным текстом
- Просмотр списка задач: `/todo`
- Отметить выполненной / вернуть обратно (inline-кнопки)
- Редактирование и удаление задач (inline-кнопки)
- Админ-режим:
  - Команда `/users` показывает список `user_id`, которые уже использовали бота

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Создайте файл `.env` в корне проекта (см. `.env.example`):

- `BOT_TOKEN` — токен вашего бота
- `ADMIN_ID` — ваш Telegram user id (числом)

## Запуск

```bash
python bot.py
```

## Хранение данных

- Задачи пользователей сохраняются в `data/<user_id>.data`
- Список пользователей, которые использовали бота: `data/users.json`

Папка `data/` исключена из git (см. `.gitignore`).
